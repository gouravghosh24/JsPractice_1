Object.create()


object dot create is a method creates a new object using an existing object as a prototype of the newly created object

Here we have an object named students and we are creating new object of students.
Like there is :
  • UnderGrads Students
  • Graduated Students
    • Using the existing object prototype we can create multiple Objects of the same nature.

    See How it Works

    You will find the code below
    const students = {
    Candidates : function(){
    document.write(this.name);
    document.write(this.age);
    document.write(this.marks);
    }
    }
    const UnderGrads = Object.create(students)
    UnderGrads.name = "Rahul Sethia"
    UnderGrads.age = 18
    UnderGrads.marks = 80
    const Graduated = Object.create(students)
    Graduated.name= "Himesh Singh"
    Graduated.age = 22
    Graduated.marks = "8.9 CGPA"
